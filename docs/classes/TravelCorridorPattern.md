# ITS Ontology - Transport Network - Travel Corridor Pattern

Travel corridor concepts that are derived from ITS Location features and can be presented with multiple geometries.

This pattern imports the following files:

- [https://w3id.org/itsdata/transport-network/v1/TransportNetworkPattern](https://w3id.org/itsdata/transport-network/v1/TransportNetworkPattern)

This pattern consists of the following classes:

- [Travel Corridor](TravelCorridor.md)
- [Travel Corridor Link](TravelCorridorLink.md)
- [Travel Corridor Segment](TravelCorridorSegment.md)

The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](../travel-corridor-pattern.ttl) and the SHACL [restrictions](../travel-corridor-shacl.ttl).
