# ITS Ontology - Transport Network - Rail Network Pattern

This pattern defines rail network concepts, which are derived from ITS Location features and can be presented with multiple geometries.

This pattern imports the following files:

- [https://w3id.org/itsdata/transport-network/v1/TransportNetworkPattern](https://w3id.org/itsdata/transport-network/v1/TransportNetworkPattern)

This pattern consists of the following classes:

- [Rail Corridor](RailCorridor.md)
- [Rail Network](RailNetwork.md)
- [Rail Section](RailSection.md)
- [Track Link](TrackLink.md)
- [Track Segment](TrackSegment.md)

The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](../rail-network-pattern.ttl) and the SHACL [restrictions](../rail-network-shacl.ttl).
