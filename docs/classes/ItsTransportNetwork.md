# ITS Ontology for Transport Network

Ontology that imports the transport-network core pattern module and the CDM-aligned transport sub-pattern modules (road, rail, pedestrian, micromobility, public transport, travel corridor, alerts). All transport network elements are grounded as geo:Feature via imports of the ITS Location ontology.

This pattern imports the following files:

- [https://w3id.org/itsdata/transport-network/v1/MicromobilityNetworkPattern](https://w3id.org/itsdata/transport-network/v1/MicromobilityNetworkPattern)
- [https://w3id.org/itsdata/transport-network/v1/PedestrianNetworkPattern](https://w3id.org/itsdata/transport-network/v1/PedestrianNetworkPattern)
- [https://w3id.org/itsdata/transport-network/v1/PublicTransportSystemPattern](https://w3id.org/itsdata/transport-network/v1/PublicTransportSystemPattern)
- [https://w3id.org/itsdata/transport-network/v1/RailNetworkPattern](https://w3id.org/itsdata/transport-network/v1/RailNetworkPattern)
- [https://w3id.org/itsdata/transport-network/v1/RoadNetworkPattern](https://w3id.org/itsdata/transport-network/v1/RoadNetworkPattern)
- [https://w3id.org/itsdata/transport-network/v1/TransportAlertPattern](https://w3id.org/itsdata/transport-network/v1/TransportAlertPattern)
- [https://w3id.org/itsdata/transport-network/v1/TransportNetworkPattern](https://w3id.org/itsdata/transport-network/v1/TransportNetworkPattern)
- [https://w3id.org/itsdata/transport-network/v1/TravelCorridorPattern](https://w3id.org/itsdata/transport-network/v1/TravelCorridorPattern)

This pattern does not contain any classes.

The formal definition of this pattern is available in [TURTLE Syntax](../its-tn.ttl).
