# ITS Ontology - Transport Network - Transport Alert Pattern

Transport alert concepts that are relevant to the transport network.

This pattern imports the following files:

- [https://w3id.org/itsdata/transport-network/v1/TransportNetworkPattern](https://w3id.org/itsdata/transport-network/v1/TransportNetworkPattern)

This pattern consists of the following classes:

- [Transport Alert](TransportAlert.md)
- [Transport Alert Thing](TransportAlertThing.md)

The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](../transport-alert-pattern.ttl) and the SHACL [restrictions](../transport-alert-shacl.ttl).
