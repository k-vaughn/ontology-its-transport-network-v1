# ITS Ontology - Transport Network - Micromobility Network Pattern

This pattern defines micromobility network concepts, which are derived from ITS Location features and can be presented with multiple geometries.

This pattern imports the following files:

- [https://w3id.org/itsdata/transport-network/v1/TransportNetworkPattern](https://w3id.org/itsdata/transport-network/v1/TransportNetworkPattern)

This pattern consists of the following classes:

- [Micromobility Lane](MicromobilityLane.md)
- [Micromobility Link](MicromobilityLink.md)
- [Micromobility Network](MicromobilityNetwork.md)
- [Micromobility Path](MicromobilityPath.md)
- [Micromobility Path Section](MicromobilityPathSection.md)
- [Micromobility Path Segment](MicromobilityPathSegment.md)

The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](../micromobility-network-pattern.ttl) and the SHACL [restrictions](../micromobility-network-shacl.ttl).
