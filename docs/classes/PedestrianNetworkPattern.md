# ITS Ontology - Transport Network - Pedestrian Network Pattern

This pattern defines pedestrian network concepts, which are derived from ITS Location features and can be presented with multiple geometries.

This pattern imports the following files:

- [https://w3id.org/itsdata/transport-network/v1/TransportNetworkPattern](https://w3id.org/itsdata/transport-network/v1/TransportNetworkPattern)

This pattern consists of the following classes:

- [Footpath](Footpath.md)
- [Footpath Lane](FootpathLane.md)
- [Footpath Link](FootpathLink.md)
- [Footpath Network](FootpathNetwork.md)
- [Footpath Section](FootpathSection.md)
- [Footpath Segment](FootpathSegment.md)

The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](../pedestrian-network-pattern.ttl) and the SHACL [restrictions](../pedestrian-network-shacl.ttl).
