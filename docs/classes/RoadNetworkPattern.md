# ITS Ontology - Transport Network - Road Network Pattern

This pattern defines road network concepts, which are derived from ITS Location features and can be presented with multiple geometries.

This pattern imports the following files:

- [https://w3id.org/itsdata/transport-network/v1/TransportNetworkPattern](https://w3id.org/itsdata/transport-network/v1/TransportNetworkPattern)

This pattern consists of the following classes:

- [Road](Road.md)
- [Road Lane](RoadLane.md)
- [Road Link](RoadLink.md)
- [Road Link User Type](RoadLinkUserType.md)
- [Road Network](RoadNetwork.md)
- [Road Section](RoadSection.md)
- [Road Segment](RoadSegment.md)
- [Vehicle Flow Rate](VehicleFlowRate.md)

The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](../road-network-pattern.ttl) and the SHACL [restrictions](../road-network-shacl.ttl).
